# 使用 Windows 系统经验

## 查看当前操作的文件被哪个进程占用

在删除或移动文件时，提示文件被占用，可以使用此方法解决。

1. 任务管理器 --> 性能 --> 打开资源监视器

    ![任务管理器](./任务管理器.png)
2. 关联的句柄里面输入操作的文件名

    ![资源监视器](./资源监视器.png)
3. 查看对应的进程，找到对应的进程，右键结束进程即可
